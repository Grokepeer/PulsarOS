### Licensing

This repository contains code generated using the STM32CubeMX and MDK-ARM toolchain. That code remains under the original licenses. The code added to that, mostly limited to the [custom](./Custom) folder, is released under the GNU-Affero-GPL-V3 license.

## Repository Structure

The folder structure is generated by the STM32CubeMX [Pulsar.ioc](./Pulsar.ioc) project for the MDK-ARM toolchain.

- [Core](./Core/) contains auto generated code
- [Custom](./Custom/) contains all the Pulsar-specific code that is written ad-hoc
- [Drivers](./Drivers/) contains code from the CMSIS and HAL code libraries
- [MDK-ARM](./MDK-ARM/) contains the MDK-ARM project files
- [Middlewares](./Middlewares/) contains the FreeRTOS-kernel code
- [Modules](./Modules/) contains the code for the radio modules
